# Earliest Deadline First

Use the following command to compile in GCC.

```
$ gcc main.c func_def.c  -o EDF
```

### Sample

| Tasks | Arrival	| Execution  |  Deadline | Period |  
|:-----:|:---------:|:----------:|:---------:|:-------:
|T1	    |   1		|	  3 	 |	6        |   9    |
|T2	    |   3		|	  2 	 |	5        |   12   |
|T3	    |   4		|	  4 	 |	10       |   16   |
